# Keystroke Logging Demonstration Using Python GUI

## Project Description
This project demonstrates a basic keylogger implemented using Python for educational and learning purposes.
It captures keystrokes entered by the user, stores them in a text file and JSON file, and provides a simple
Graphical User Interface (GUI) to control the keylogging process.

⚠️ This project is strictly for educational purposes only.

---

## Features
- Real-time keystroke capture
- Logs stored in text and JSON format
- Simple Python GUI using Tkinter
- Start button to initiate keylogging

---

## Technologies Used
- Python
- Tkinter (GUI)
- Pynput (Keyboard Listener)
- File Handling
